Detailed Description

This file contains one possible solution to the first RT1 assignment.

Author
Emanuele Rambaldi
Version
0.1
Date
5/11/2021

This assignment aim is to guide the robot shown in the simulation through the established path, grabbing the silver tokens that it ecounters and moving them behind it and avoiding collision with golden tokens.

def assignment1.main ( )

Function that calls all the other ones.

Parameters
Noparameters
Returns
No return values

By calling the other functions, the main function guide the robot along the path defined by the golden blocks in counter-clockwise direction, making sure that it does not collide with them. Furthermore it makes the robot detect the silver blocks that it encounters along the way, makes it grab them and move them behind itself.
